# Preview Screenshots

Marketplace-Pflicht: **mindestens 1 Preview-Bild** muss hier liegen, bevor du in der Maker Console submittest.

## Was rein soll

- **Mindestens 1 Screenshot** des Plugins im Einsatz (PNG oder JPG)
- Sinnvoll: 2–3 Bilder, z. B.
  - `01-encoder.png` — Stream Deck+ Encoder mit Cover, Titel/Artist, Volume-Bar
  - `02-key.png` — Keypad-Taste mit Cover als Hintergrund + Titel
  - `03-marquee.png` — langer Track-Name mit Laufschrift

## Wie aufnehmen

1. Stream Deck Software öffnen, Action auf SD+ ziehen
2. Apple Music starten, Track mit prominentem Cover + Titel abspielen
3. Foto vom physischen SD+ Display (Smartphone-Kamera reicht — gut beleuchten) **oder** Screenshot aus dem Stream-Deck-Software-Preview
4. Dateien hier ablegen, namentlich `01-…`, `02-…` für Reihenfolge im Marketplace

## Format-Empfehlungen

Elgato dokumentiert keine harten Pixel-Vorgaben für Gallery-Items, in der Praxis bewährt:

- 1920×1080 (Landscape) oder 1080×1080 (Square) als Mindestauflösung
- PNG für Screenshots mit klaren Kanten, JPG für Fotos
- Keine Wasserzeichen, keine Werbung, keine Markenlogos außer dem eigenen
